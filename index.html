just-install

The humble package installer for Windows

Installation

Run this in cmd.exe, as Administrator:

msiexec.exe /i http://go.just-install.it

What can we do?


Install a package

just-install firefox

List Available Packages

just-install list

Install for specific architecture

just-install -a x86 go

Help!

just-install --help
